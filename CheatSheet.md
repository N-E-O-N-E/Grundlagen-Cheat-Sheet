# Cheat Sheet

### `var`

Erstellt eine Variable, die sich 채ndern kann

```swift
var age = 25
```
---
### `let`

Erstellt eine Konstante, die sich nicht 채ndern kann

```swift
let name = "John"
```
---
### `if`

F체hrt Code aus, wenn eine Bedingung wahr ist

```swift
if (age > 18) {
    print("Adult")
}
```
---
### `else`

F체hrt Code aus, wenn die Bedingung falsch ist

```swift
if (age > 18) {
    print("Adult")
} else {
    print("Minor")
}
```
